\doxysubsection{Detailed Description}
Shell command interface for the Nucleo-\/\+STM32\+G474 microcontroller. 

Provides a basic shell interface to execute commands via UART. Commands include setting PWM parameters, starting/stopping PWM, resetting the microcontroller, and measuring current.
